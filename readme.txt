Simple form to collect payment with Venmo (https://venmo.com).

== Description ==

The plugin will allow user to embed simple static form to their page/post to collect payments with Venmo (https://venmo.com). To embed the form user must use shortcode generated by the plugin page in the admin dashboard. Plugin does NOT require any personal and/or sensitive information, like ex. password to Venmo account. Plugin require only the link to the Venmo's public user profile.
Users either can embed form with 2 input fields and let users to enter any amount and any description OR admins can save static amounts and embed the form with the amounts in the dropdown.

== Installation ==

1. Upload the plugin files to the /wp-content/plugins/plugin-name directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the Plugins screen in WordPress
3. As an admin go to the Wordpress dashboard and Click Pay with Venmo
4. Add link to your Venmo account.
5. Copy shortcode and paste on any page or post where you would like to collect your payments.
